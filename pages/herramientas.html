Herramientas Utiles


Aquí vas a encontrar algunas herramientas simples pero útiles que te van a dar información de lo que está pasando en tu dispositivo, si estamos actualizados con los últimos parches de seguridad para nuestro Sistema operativo y la base de datos de antivirus actualizada, si hay alguna aplicación de la que no estemos enterados que está corriendo en nuestro dispositivo y puede comprometer nuestra seguridad.


Herramienta Privilegios Instalacion Fiabilidad
Inspector de Red Usuario No requiere Preciso
Inspector de IP Usuario No requiere Preciso
Inspector de Antivirus Usuario Requerido Aproximado
Inspector de Parches Usuario Requerido Aproximado
Inspector de Servicios Usuario Requerido Bajo

Inspector de red


Nuestro inspector de Red, es un script que va a ejecutar un comando en nuestro sistema operativo para obtener todas las direcciones IP con la que nuestro computador se esta comunicando, estas direcciones IP, serán enviadas a nuestro servicio para ser comparadas con varias bases de datos que mantienen un listado de Direcciones IP catalogadas como maliciosas

Inspector de IP


Nuestro inspector de IP, obtendrá la dirección IP pública que tienes asignada y hará una verificación en las bases de datos de IP reportadas, y verificará si la tuya esta dentro de este listado. Con esta confirmación deberás hacer una revisión de tu computador con un profesional y comprobar si tu dispositivo esta siendo utilizado para alguna actividad maliciosa sin tu conocimiento

Inspector de Antivirus


Nuestro Inspector de antivirus, hará una consulta al sistema operativo de tu computadora para obtener la información de versión y parche de antivirus que tengas instalado, y hará una comparación con la publicada por el fabricante de antivirus que tengas, con lo cual verificara si se encuentran actualizados o no

Inspector de parches de Seguridad


Nuestro inspector de parches de seguridad, realizara una verificación dentro del computador para obtener la información de que nivel de parches se encuentra nuestro dispositivo, y hará una comparación con la base de datos del fabricante para verificar si se encuentran actualizado o no

Inspector de Servicios de windows


Nuestro inspector de Servicios de windows es una herramienta que hará una revisión a los servicios que están ejecutándose en tu computador, y hará una busqueda para comparar si alguno de estos servicios se corresponde con algún programa malicioso

(Nota: Muchos de los programas maliciosos ocultan su actividad bajo nombres de servicios validos, por lo que no podrán ser detectados por nuestra herramienta, estamos trabajando para mejorar)
